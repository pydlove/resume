主题： 安装MarkdownPad2

1、下载地址如下：

https://markdownpad.com/download.html

2、正常安装即可

3、安装完成会出现一个错误：Markdown : An error occurred with the Html rendering component
其实就是少一个东西 没有安装 awesomium_v1.6.6_sdk_win md右侧无法实时显示内容

下载地址如下：

http://markdownpad.com/download/awesomiumv1.6.6sdk_win.exe

4、下载后正常安装后重启MarkdownPad2即可
